# Scramblies challenge

### Run service

```bash
./run.sh
```

Go to browser and open http://localhost:2019

### Run tests

```bash
cd server && lein test
```

### Development

Run server

```bash
cd server && lein repl
```

Run client

```bash
cd client && lein clean && lein figwheel dev
```

Go to browser and open http://localhost:4334
